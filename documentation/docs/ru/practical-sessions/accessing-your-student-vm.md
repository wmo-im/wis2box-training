# Доступ к вашей студенческой виртуальной машине

!!! abstract "Учебные результаты"

    По окончании этой практической сессии вы сможете:

    - получить доступ к вашей студенческой виртуальной машине через SSH и WinSCP
    - проверить наличие необходимого программного обеспечения для практических упражнений
    - убедиться, что у вас есть доступ к учебным материалам для этого тренинга на вашей локальной студенческой виртуальной машине

## Введение

В рамках локальных тренинговых сессий wis2box вы можете получить доступ к вашей личной студенческой виртуальной машине в локальной учебной сети под названием "WIS2-training".

На вашей студенческой виртуальной машине предустановлено следующее программное обеспечение:

- Ubuntu 22.0.4.3 LTS [ubuntu-22.04.3-live-server-amd64.iso](https://releases.ubuntu.com/jammy/ubuntu-22.04.3-live-server-amd64.iso)
- Python 3.10.12
- Docker 24.0.6
- Docker Compose 2.21.0
- Текстовые редакторы: vim, nano

!!! note

    Если вы хотите провести этот тренинг вне локальной учебной сессии, вы можете использовать свой собственный экземпляр с помощью любого облачного провайдера, например:

    - GCP (Google Cloud Platform) VM instance `e2-medium`
    - AWS (Amazon Web Services)  ec2-instance `t3a.medium` 
    - Azure (Microsoft) Azure Virtual Machine `standard_b2s`

    Выберите Ubuntu Server 22.0.4 LTS в качестве ОС.
    
    После создания вашей виртуальной машины убедитесь, что у вас установлены python, docker и docker compose, как описано на [wis2box-software-dependencies](https://docs.wis2box.wis.wmo.int/en/latest/user/getting-started.html#software-dependencies).
    
    Архив с релизом wis2box, используемым в этом тренинге, можно скачать следующим образом:

    ```bash
    wget https://github.com/World-Meteorological-Organization/wis2box-release/releases/download/1.0.0/wis2box-setup.zip
    unzip wis2box-setup.zip
    ```
    
    Вы всегда можете найти последний архив 'wis2box-setup' по адресу [https://github.com/World-Meteorological-Organization/wis2box/releases](https://github.com/World-Meteorological-Organization/wis2box-release/releases).

    Учебные материалы, используемые в этом тренинге, можно скачать следующим образом:

    ```bash
    wget https://training.wis2box.wis.wmo.int/exercise-materials.zip
    unzip exercise-materials.zip
    ```

    Для выполнения учебных материалов требуются следующие дополнительные пакеты Python:

    ```bash
    pip3 install minio
    pip3 install pywiscat==0.2.2
    ```

    Если вы используете студенческую виртуальную машину, предоставленную во время локальных тренинговых сессий WIS2, необходимое программное обеспечение уже будет установлено.

## Подключение к вашей студенческой виртуальной машине в локальной учебной сети

Подключите ваш ПК к локальному Wi-Fi, транслируемому в помещении во время тренинга WIS2, согласно инструкциям, предоставленным тренером.

Используйте SSH-клиент для подключения к вашей студенческой виртуальной машине, используя следующие данные:

- **Host: (предоставлено во время очного обучения)**
- **Port: 22**
- **Username: (предоставлено во время очного обучения)**
- **Password: (предоставлено во время очного обучения)**

!!! tip
    Обратитесь к тренеру, если вы не уверены в имени хоста/имени пользователя или у вас возникли проблемы с подключением.

После подключения, пожалуйста, измените ваш пароль, чтобы другие не могли получить доступ к вашей виртуальной машине:

```bash
limper@student-vm:~$ passwd
Changing password for testuser.
Current password:
New password:
Retype new password:
passwd: password updated successfully
```

## Проверка версий программного обеспечения

Чтобы иметь возможность запускать wis2box, на студенческой виртуальной машине должны быть предустановлены Python, Docker и Docker Compose. 

Проверьте версию Python:
```bash
python3 --version
```
возвращает:
```console
Python 3.10.12
```

Проверьте версию Docker:
```bash
docker --version
```
возвращает:
```console
Docker version 24.0.6, build ed223bc
```

Проверьте версию Docker Compose:
```bash
docker compose version
```
возвращает:
```console
Docker Compose version v2.21.0
```

Чтобы убедиться, что ваш пользователь может выполнять команды Docker, ваш пользователь был добавлен в группу `docker`. 

Чтобы проверить, что ваш пользователь может запустить docker hello-world, выполните следующую команду:
```bash
docker run hello-world
```

Это должно загрузить образ hello-world и запустить контейнер, который выводит сообщение. 

Убедитесь, что вы видите следующее в выводе:

```console
...
Hello from Docker!
This message shows that your installation appears to be working correctly.
...
```

## Осмотр учебных материалов

Осмотрите содержимое вашего домашнего каталога; это материалы, используемые в рамках тренинга и практических сессий.

```bash
ls ~/
```
возвращает:
```console
exercise-materials  wis2box
```

Если на вашем локальном ПК установлен WinSCP, вы можете использовать его для подключения к вашей студенческой виртуальной машине и просмотра содержимого вашего домашнего каталога, а также для загрузки или выгрузки файлов между вашей виртуальной машиной и вашим локальным ПК. 

WinSCP не требуется для тренинга, но может быть полезен, если вы хотите редактировать файлы на вашей виртуальной машине с помощью текстового редактора на вашем локальном ПК.

Вот как вы можете подключиться к вашей студенческой виртуальной машине с помощью WinSCP:

Откройте WinSCP и нажмите на "New Site". Вы можете создать новое SCP-подключение к вашей виртуальной машине следующим образом:

<img alt="winscp-student-vm-scp.png" src="/../assets/img/winscp-student-vm-scp.png" width="400">

Нажмите 'Save' и затем 'Login' для подключения к вашей виртуальной машине.

И вы должны увидеть следующее содержимое:

<img alt="winscp-student-vm-exercise-materials.png" src="/../assets/img/winscp-student-vm-exercise-materials.png" width="600">
