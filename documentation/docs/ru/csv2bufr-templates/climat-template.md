# Шаблон csv2bufr для ежедневных климатических данных (CLIMAT)

Сообщения **CLIMAT** содержат ежемесячные климатические сводки, составленные на основе ежедневных наблюдений на синоптических и климатологических станциях, для поддержки мониторинга климата, исследований и архивирования.

Шаблон CLIMAT предоставляет стандартизированный формат CSV для создания сообщений CLIMAT, закодированных в формате BUFR для последовательностей 301150,307073.
