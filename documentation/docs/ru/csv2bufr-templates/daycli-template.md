# Шаблон csv2bufr для ежедневных климатических данных (DAYCLI)

Шаблон **DAYCLI** предоставляет стандартизированный формат CSV для преобразования ежедневных климатических данных в последовательность BUFR 307075.

Этот формат предназначен для использования с системами управления климатическими данными (Climate Data Management Systems) для публикации данных в WIS2, поддерживая требования к отчетности по ежедневным климатическим наблюдениям.

Этот шаблон отображает ежедневные наблюдения за:

 - Минимальной, максимальной и средней температурой за 24-часовой период
 - Общим накопленным количеством осадков за 24-часовой период
 - Общей глубиной снежного покрова на момент наблюдения
 - Глубиной свежевыпавшего снега за 24-часовой период

Этот шаблон требует дополнительной метаинформации по сравнению с упрощенным шаблоном AWS: метод расчета средней температуры; высота датчиков и станций; классификация условий размещения и качества измерений.

!!! Note "О шаблоне DAYCLI"
    Обратите внимание, что последовательность BUFR для DAYCLI будет обновлена в 2026 году для включения дополнительной информации и пересмотренных флагов контроля качества (QC). Шаблон DAYCLI, включенный в wis2box, будет обновлен, чтобы отразить эти изменения. ВМО сообщит, когда программное обеспечение wis2box будет обновлено для включения нового шаблона DAYCLI, чтобы пользователи могли соответствующим образом обновить свои системы.
